Name

disconnect

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MServer server;
MClient client;

String message;

void setup()
{
	server = MBluetooth.startService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	client = server.available();
	
	if(client != null)
	{
		message = client.readString();

		if(message.equals("Bye"))
			server.disconnect(client);
	}
}

Description

Disconnect a particular client.

Syntax

  • server.disconnect(client)

Parameters

  • server
    • Any variable of type MServer
  • client
    • MClient : the client to disconnect
