Name

write

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MClient client;

int val;

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	if(client != null)
		client.write(val);
}

Description

Write data to a server

Syntax

  • client.write(data)

Parameters

  • client
    • Any variable of type MClient
  • data
    • int,char,bytes[],String : The data to send
