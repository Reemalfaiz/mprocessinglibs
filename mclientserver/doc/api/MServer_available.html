Name

available

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MServer server;
MClient client;

int val = 0;

void setup()
{
	server = MBluetooth.startService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	client = server.available();
	
	if(client != null)
		val = client.read();
}

Description

Returns the next client in line with a new message.

Syntax

  • server.available()

Parameters

  • server
    • Any variable of type MServer.

Returns

  • MClient
    • Client that sends the message.
