Name

stop

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MServer server;

int val = 0;

void setup()
{
	server = MBluetooth.startService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	if(server != null)
		server.write(val);
}

void keyPressed()
{
	if(key == '1')
	{
		server.stop();
		server = null;
	}
}

Description

Disconnects all clients and stops the server.

Syntax

  • server.stop()

Parameters

  • server
    • Any variable of type MServer
