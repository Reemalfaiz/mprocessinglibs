Name

readChar

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MClient client;

int val = 0;

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void draw()
{
	background(val);

	if(client != null && client.available() > 0)
		println("" + client.readChar());
}

Description

Returns the next byte in the buffer as a char. Returns -1 if nothing is there.

Syntax

  • client.readChar()

Parameters

  • client
    • any variable of type Client

Returns

  • char
    • next byte as a char.
