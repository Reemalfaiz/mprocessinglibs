Name

write

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MServer server;

int val = 0;

void setup()
{
	server = MBluetooth.startService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	if(server != null)
		server.write(val);
}

Description

Write data to all connected clients

Syntax

  • server.write(data)

Parameters

  • server
    • Any variable of type Server
  • data
    • int,char,bytes[],String : The data to send
