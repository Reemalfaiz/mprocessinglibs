Name

MClient

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MClient client;

int val = 0;

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void draw()
{
	if(client != null && client.available())
		val = client.read();

	background(val);
}

Description

A client connects to a server and sends data back and forth. If anything goes wrong with the connection, for example the host is not there or is listening on a different port, an exception is thrown. The client is created by the service that provides client server communication, for example bluetooth.

Methods

  • available()
    • Returns the number of bytes in the buffer waiting to be read
  • read()
    • Reads the next byte
  • readChar()
    • Returns the next byte as a char
  • readBytes()
    • Reads some number of bytes from the input
  • readString()
    • Reads a string
  • write()
    • Writes bytes, chars, ints, bytes[], Strings
  • stop()
    • Disconnects from the server
