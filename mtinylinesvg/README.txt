MTinyLineSVG Library for Mobile Processing
------------------------------------------

MTinyLineSVG library use to SVG Tiny rendering, full support of animations 
and events.

The examples directory contains examples to show library features.

Installation :
--------------

- Copy this library to Mobile Processing libraries directory
