MPIM Library for Mobile Processing
-----------------------------------

This library provides access to the Personal Information storage in the phone 
through the JSR 75 API PIM Optional Package 1.0

This initial release allow the access to the contact list and the creation of 
new contacts.

The examples directory contains examples to show library features.

Enjoy!
