MSound Library for Mobile Processing
------------------------------------

MSound library allows Mobile Processing to load and play sound files from 
the data directory or from a server, record captured audio and use midi characteristics.

This Library is based on the class and methods availables under Processing 
to allow an easy understading.

The examples directory contains examples to show library features.

Installation :
--------------

- Copy this library to Mobile Processing libraries directory
