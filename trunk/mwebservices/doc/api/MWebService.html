Name

MWebService

Examples

import mjs.processing.mobile.mwebservices.*;

MWebService wService;

String endPoint = "http://services.xmethods.net/soap";
String namespace = "urn:xmethods-delayed-quotes";
String method = "getQuote";

void setup()
{
  noLoop();
}

void keyPressed()
{
  new Thread()
  {
    public void run()
    {
      wService = new MWebService(endPoint,namespace,method);
      wService.addParameter("symbol","sunw");
      
      println("Return : " + wService.call());
    }
  }.start();
}

Description

Datatype to make calls to webservices.

Methods

  • addParameter()
    • Adds a parameter (property) to the call.
  • call()
    • Returns the result of calling the webservice.

Constructors

  • MWebService(endPointURL,namespace,methodName)

Parameters

  • endPointURL
    • String : The destination to POST SOAP data.
  • namespace
    • String : The namespace for the soap object.
  • methodName
    • String : The name for the soap object
