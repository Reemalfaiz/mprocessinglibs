MTinyLine2d Library for Mobile Processing
-----------------------------------------

MTinyLine2d library use the 2D render engine to provide better 
graphic manipulation in Mobile Processing.

The examples directory contains examples to show library features.

Installation :
--------------

- Copy this library to Mobile Processing libraries directory
