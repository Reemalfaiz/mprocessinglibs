MNokiaUI Library for Mobile Processing
--------------------------------------

This library provides implementation of Mobile Processing primitives to 
support NokiaUI API.

The current version adds alpha channel color support

The examples directory contains examples to show library features.

Requirements :
--------------

To use the library you need to hack the Mobile Processing Core, this tasks
is described below

Also, you need a emulator and phone with support of NokiaUI API.

The Sun Wireless Toolkit 2.2, 2.3, 2.5 don't support the NokiaUI API

The following SDKs supports NokiaUI API :

- The Nokia Prototype SDK 4.0 Beta for JME and others Nokia SDKs
  http://www.forum.nokia.com/

The following page shows the devices that support NokiaUI API

	http://www.j2mepolish.org/devices/devices-nokia-ui.html
	

This library was tested with the Nokia Prototype SDK 4.0 Beta for JME	


Installation :
--------------

- Copy this library to Mobile Processing libraries directory
