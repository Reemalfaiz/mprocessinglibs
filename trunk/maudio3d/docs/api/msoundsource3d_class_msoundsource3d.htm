MAudio3D index
Name MSoundSource3D
Examples 
None available
Description Represents a sound source in a virtual acoustical space.
Constructors 
MSoundSource3D(locator);
MSoundSource3D(locator, pMIDlet);
Parameters
locator   A locator string in URI syntax that describes the media content or resource route.
pMIDlet   The current midlet.
Methods
directivity ( )   Gets the directivity pattern of a sound source.
distanceAttenuation ( )   Sets all the 3D audio distance attenuation parameters simultaneously. Distances are specified in millimeters.
location ( )   Gets the coordinates of the current location.
spherical ( )   Moves the object to the new location. Sets the new location using spherical coordinates. The negative z-axis is the reference. That is, a location where both azimuth and elevation are zero, is on the negative z-axis. Radius is defined in millimeters.
