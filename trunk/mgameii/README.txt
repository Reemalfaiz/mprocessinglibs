MGame Library for Mobile Processing
-----------------------------------

MGame enable the development of rich gaming content for wireless devices.

The examples directory contains examples to show library features.

Installation :
--------------

- Copy this library to Mobile Processing libraries directory 
- Don't forget that in Preferences shall to be MDIP 2.0

Enjoy!
