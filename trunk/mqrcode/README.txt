MQRCode Library for Mobile Processing
-------------------------------------

MQRCode Library allows encode and decode QRCodes 

This library is based on : 

- QRCode decode library for J2ME 0.9
  http://sourceforge.jp/projects/qrcode

- QRCode encode library 0.50 beta 10
  http://www.swetake.com/qr/java/qr_java.html

The encode method encode an string in a QRCode version 3 using M error 
correction.

The source directory contains the source code for each library, the 
com.swetake.util.Qrcode class has some changes to run in J2ME

The examples directory contains examples to show library features.
