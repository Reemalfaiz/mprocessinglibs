MClientServer Library

The MClientServer library makes it easy to read and write data across mobile devices on a TCP/IP or Bluetooth network. It allows the creation of clients ands servers. A server connects to a list of clients for reading and writing data. A client is able to read and write data to a server.

This library gives support to other libraries like Bluetooth to create his own services.

MServer

The server class is used to create server objects which send and receive data to and from its associated clients (other programs connected to it)

methods
