Name

readBytes

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;


byte[] info = new byte[512];

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void keyPressed()
{
	if(client != null && client.available() > 0)
		client.readBytes(info);
}

Description

Reads some number of bytes from the input into the array.

Syntax

  • client.readBytes()

Parameters

  • client
    • any variable of type MClient

Returns

  • byte[]
    • array into which the data is read
