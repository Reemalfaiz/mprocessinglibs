Name

available

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MClient client;

int val = 0;

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void draw()
{
	background(val);

	if(client != null && client.available() > 0)
		val = client.read();
}

Description

Returns the number of bytes available.

Syntax

  • client.available()

Parameters

  • client
    • any variable of type MClient

Returns

  • int
    • Number of bytes availables or -1 if any can be read.
