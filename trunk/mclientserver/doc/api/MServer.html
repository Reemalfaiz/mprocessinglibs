Name

MServer

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MServer server;

int val = 0;

void setup()
{
	server = MBluetooth.startService(0x1234567890L);
}

void draw()
{
	val = (val + 1) % 255;
	background(val);

	if(server != null)
		server.write(val);
}

Description

A server sends and receives data to and from its associated clients (other programs connected to it). When a server is started, it begins to listening for connections on the specified address. The server is created by the service that provides client server communication, for example bluetooth.

Methods

  • write()
    • Write data to all connected clients
  • available()
    • Returns the next client in line with a new message
  • stop()
    • Disconnect all clients and stops the server
  • disconnect()
    • Disconnect a particular client
