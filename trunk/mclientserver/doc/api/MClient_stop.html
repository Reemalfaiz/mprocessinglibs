Name

stop

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

MClient client;

int val = 0;

void setup()
{
	client = MBluetooth.selectService(0x1234567890L);
}

void draw()
{
	background(val);

	if(client != null && client.available() > 0)
		val = client.read();
}

void keyPressed()
{
	if(key == '1')
	{
		client.stop();
		client = null;
	}
}

Description

Disconnects the client from the server.Use to shut the connection when you're finished with the Client.

Syntax

  • client.stop()

Parameters

  • client
    • Any variable of type MClient
