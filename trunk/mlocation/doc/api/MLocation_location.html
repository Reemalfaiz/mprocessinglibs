Name

location

Examples

import mjs.processing.mobile.mlocation.*;
				
double[] coords = new double[2];
MLocation.location(coords);

int latitude = (int) (coords[0] * 10000);
int longitude = (int) (coords[1] * 10000);

text("Latitude : " + latitude,10,10);
text("Longitude : " + longitude,10,20);

Description

Fills the array with the current values of latitude and longitude.

Syntax

  • MLocation.location()

Parameters

  • Coords
    • double[] : Array of two doubles where the latitude and longitude will be store.
