Name @related
Example 
@related HtmlElement
@related addAttribute ( )
Description You can insert links to related parts of the documentation, that means to fields and methods of the same class and to other classes. When you insert a link you have to write the exact name of the class, field or method. If it is different proDOC gives an error. You have to put round brackets after links to methods.
Usage optional
Used by class description
field description
method description
