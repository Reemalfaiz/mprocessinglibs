Name

nearDevices

Examples

import mjs.processing.mobile.mbluetooth.*;

void keyPressed()
{
	// We need to create a different thread to avoid deadlocks
	new Thread()
	{
		public void run()
		{
			MDevice[] devices = MBluetooth.nearDevices();
	
			for(int i=0; i<devices.length; i++)
				println(devices[i].name());
		}
	}.start();
}

Description

Returns an array with all the devices near to the local device.

Syntax

  • MBluetooth.nearDevices()

Returns

  • Device[]
    • Array with all the near devices
