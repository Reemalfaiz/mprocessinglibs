Name

startDefaultService

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

int action;

boolean serverRunning;

MServer server;

void draw()
{
	if(action == 1 && !serverRunning)
	{
		startServer()
		serverRunning = true;
	}
}

void startServer()
{
	server = MBluetooth.startDefaultService();
}

void keyPressed()
{
	if(key == '*')
		action = 1;
		
	if(server != null)
		server.writeChar(key);
}

Description

Start the default Mobile Processing Service.

See the ClientSever documentation for more information about data exchange

Syntax

  • MBluetooth.startDefaultService()

Returns

  • MServer
    • A server ready to accept clients and send data
