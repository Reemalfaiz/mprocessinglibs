Name

MBluetooth

Examples

import mjs.processing.mobile.mbluetooth.*;

String name;

void setup()
{
	name = MBluetooth.deviceName();
}

void draw()
{
	textMode(CENTER);
	text(name,width/2,heigth/2);
}

Description

Basic class to access Bluetooth features.

Methods

  • start()
    • Start the bluetooth service making the device discoverable.
  • deviceName()
    • Returns the friendly name of the local device.
  • nearDevices()
    • Returns all the devices near to the local device..
  • selectService()
    • Creates a client to the specified service uuid.
  • startService()
    • Start a service with the uuid specified.
  • selectDefaultService()
    • Creates a client to the Mobile Processing Service if is available.
  • startDefaultService()
    • Start the default Mobile Processing Service.
