Name

selectDefaultService

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

int action;

boolean clientRunning;

MClent client;

void draw()
{
	if(action == 1 && !clientRunning)
	{
		startClient()
		clientRunning = true;
	}

	if(client != null && client.available())
		println("" + client.readChar());
}

void startClient()
{
	client = MBluetooth.selectDefaultService();
}

void keyPressed()
{
	if(key == '*')
		action = 1;
}

Description

Creates a client with the Mobile Processing Service identified with the uuid 0xCAFE2005

See the ClientSever documentation for more information about data exchange

Syntax

  • MBluetooth.selectDefaultService()

Returns

  • MClient
    • a client with the default service or null if service is not available.
