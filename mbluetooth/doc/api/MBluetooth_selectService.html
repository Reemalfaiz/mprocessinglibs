Name

selectService

Examples

import mjs.processing.mobile.mbluetooth.*;
import mjs.processing.mobile.mclientserver.*;

int action;
int uuid = 0x67890;

boolean clientRunning;

MClent client;

void draw()
{
	if(action == 1 && !clientRunning)
	{
		startClient()
		clientRunning = true;
	}

	if(client != null && client.available())
		println("" + client.readChar());
}

void startClient()
{
	client = MBluetooth.selectService(uuid);
}

void keyPressed()
{
	if(key == '*')
		action = 1;
}

Description

Creates a client with the uuid specified (create your own uuid number for each service), the service uuid is the same uuid used in the service server.

See the ClientSever documentation for more information about data exchange

Syntax

  • MBluetooth.selectService(uuid)

Parameters

  • uuid
    • The service uuid, you can specified in hexa, example 0x998877

Returns

  • MClient
    • a client with the default service or null if service is not available.
