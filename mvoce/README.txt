MVoce Library for Processing
----------------------------

This library allows synthesize voice and recognize speech using the Voce 
library a speec interface over the CMU Sphinx4 and FreeTTS. 

Check the examples directory to see how to use the library to convert 
text to speech or recognize words or both. 

This library is a wrapper around the Voce library that use CMU Sphinx4 and 
FreeTTS.

This library only support english language.
